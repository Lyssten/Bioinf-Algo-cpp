# Hamming Distance

## Задача

Вычислить расстояние Хэмминга между двумя строками.

**Вход:** две строки одинаковой длины  
**Выход:** число позиций, где символы различаются

**Пример:**
```
s1: GGGCCGTTGGT
s2: GGACCGTTGAC
Вывод: 3

Позиции различий:
      GGGCCGTTGGT
      || ||| |||
      GGACCGTTGAC
      ↑ ↑     ↑
      2 3     8 (0-based)
```

## Теория

### Расстояние Хэмминга
**HammingDistance(p, q)** — количество позиций, в которых соответствующие символы строк p и q различны.

**Формальное определение:**
```
HammingDistance(p, q) = |{i : p[i] ≠ q[i]}|
```

**Применение в биоинформатике:**
- Измерение эволюционной дистанции между последовательностями
- Поиск приближённых вхождений (с mismatches)
- Оценка качества секвенирования

### Ограничения
- Строки должны быть **одинаковой длины**
- Не учитывает вставки и удаления (indels)
- Для indels используется **расстояние Левенштейна**

## Алгоритм

### Идея
Попозиционное сравнение двух строк с подсчётом несовпадений.

### Псевдокод
```
HammingDistance(p, q):
    if |p| ≠ |q|:
        ошибка
    
    dist ← 0
    for i ← 0 to |p|-1:
        if p[i] ≠ q[i]:
            dist ← dist + 1
    
    return dist
```

## Реализация на C++

```cpp
int hammingDistance(const std::string& s1, const std::string& s2) {
    if (s1.size() != s2.size())  // (1)
        return -1;
    
    int dist = 0;  // (2)
    for (size_t i = 0; i < s1.size(); ++i) {  // (3)
        if (s1[i] != s2[i])  // (4)
            ++dist;
    }
    return dist;
}
```

### Особенности реализации

**1. Проверка длины**
```cpp
if (s1.size() != s2.size())
    return -1;
```
- Возврат -1 сигнализирует об ошибке
- Альтернатива: выбросить исключение `throw std::invalid_argument(...)`

**2. Тип счётчика**
```cpp
int dist = 0;
```
- `int` достаточно для типичных длин последовательностей
- Для очень длинных строк можно использовать `size_t`

**3. Цикл по размеру первой строки**
```cpp
for (size_t i = 0; i < s1.size(); ++i)
```
- После проверки длин s1.size() == s2.size()
- `size_t` для согласованности с `.size()`

**4. Сравнение символов**
```cpp
if (s1[i] != s2[i])
    ++dist;
```
- Оператор `[]` возвращает ссылку на символ
- Сравнение символов — **O(1)**

## Альтернативные реализации

**С использованием `std::mismatch` (C++14):**
```cpp
auto result = std::mismatch(s1.begin(), s1.end(), s2.begin());
int dist = std::distance(result.first, s1.end());
// Но это найдёт только первое несовпадение, нужно считать все
```

**С `std::inner_product`:**
```cpp
int dist = std::inner_product(
    s1.begin(), s1.end(), s2.begin(), 0,
    std::plus<>(),
    std::not_equal_to<>()
);
```

**С range-based for и индексом:**
```cpp
int dist = 0;
size_t i = 0;
for (char c : s1) {
    if (c != s2[i++]) ++dist;
}
```

## Запуск

```bash
# Без аргументов — тестовые данные
./hamming_distance

# Аргументы: <String1> <String2>
./hamming_distance "GGGCCGTTGGT" "GGACCGTTGAC"
# Вывод: 3
```

## Сложность

### Время: O(n)

**Обоснование:**
- Один проход по строке: **n** итераций (где n = длина строки)
- На каждой итерации:
  - Доступ по индексу `s1[i]`, `s2[i]` — **O(1)**
  - Сравнение символов — **O(1)**
  - Инкремент счётчика — **O(1)**
- Проверка длины в начале: **O(1)** (`size()` хранится в строке)
- Итого: **O(n)**

**Пример:** Две строки по 1000 символов → ~1000 сравнений

### Память: O(1)

**Обоснование:**
- Счётчик `dist`: **4 байта** (int)
- Индекс `i`: **8 байт** (size_t на 64-бит)
- Никаких дополнительных структур данных
- Итого: **O(1)** константная память

**Примечание:** Входные строки передаются по константной ссылке (`const&`), копирования нет
